Movie API Documentation

Endpoints

GET /collection/films - Get all films from collection

GET /collection/films/:programid - Get a single film by programid

GET /collection/sql/films - Get all films from sql database

GET /collection/sql/films/:programid - Get a single film by programid from sql database

GET /collection/stream/:programid - stream a film by programid

GET /collection/sql/stream/:programid - stream a film by programid from sql database

GET /live/lrt - LRT live

GET /live/plius - LRT Plius live

GET /live/lituanica - LRT lituanica live

GET /live/lr - LRT radio live

GET /live/klasika - LRT klasika live

GET /live/opus - LRT opus live

POST /collection/films - Create a new film

PUT /collection/films/:programid - Update a film by programid

DELETE /collection/films/:programid - Delete a film by programid
